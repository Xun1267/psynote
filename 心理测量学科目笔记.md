（3）客观性

第三章 经典测验理论的基本假设
1. 误差的定义
 误差是指与测量目的无关的因素所引起的一种不准确或不一致的效应。

2. 误差的种类
（1）随机误差
 由与测量目的无关的偶然因素所引起，影响准确性和一致性
（2）系统误差
 由于测量目的无关的因素所引起的恒定而有规律的效应，只影响准确性

3. 误差的来源
误差来源	具体内容
典型示例

测量工具
1. 题目取样	仅选取部分知识点作为考题（如数学测验只考代数不考几何）
	2. 题目的编制技术	题目表述模糊（如“大部分情况下”未量化）、选项设计不科学（如存在暗示性选项）
施测过程
1. 物理环境	考场嘈杂、温度过高/过低、照明不足
	2. 测试时间	时间紧张导致未完成所有题目、计时器故障延长/缩短考试时间
	3. 主试特征	主试态度严厉引发紧张、方言口音导致理解偏差
	4. 意外干扰	突然停电、考场外突发噪音（如施工声）
	5. 评分记分	主观题评分标准不一（如作文评分浮动）、机器阅卷识别错误
测量对象
1. 测验焦虑	手抖写错答案、大脑空白无法思考
	2. 测验经验	首次考试者因不熟悉流程超时、多次应试者掌握“猜题技巧”
	3. 学习、发展与教育	突击复习者短期记忆效应、不同教育背景者知识储备差异
	4. 应试动机	敷衍答题（如职业测评中故意隐藏真实倾向）、讨好主试（如选择社会赞许性答案）
	5. 练习效应	重复刷题提高分数（如托福机经训练）、题型熟悉度影响反应速度
	6. 反应方式	习惯性跳过难题、总是选择第一个看似合理的选项
	7. 生理变因	感冒头晕影响注意力、考前失眠导致思维迟缓

4. 真分数的含义
（1）理论定义：一个测量工具在测量得没有误差时所得到得纯正值
（2）操作定义：经过无数次测量所得的平均值

5. 真分数的数学模型
 X=T+E

6. 真分数的三个假设
（1）若一个人的某种心理特质可以用平行的测验反复测量足够多次，则其平均误差为0，即其是的分数的平均值会接近于真分数
（2）真分数和测量误差之间相互独立
（3）各平行测验上的误差分数之间相关为零

7. 真分数的三个结论
（1）真分数不变，测量的任务是估计真分数的大小
（2）真分数和实得分数之间存在着线性关系
（3）测量误差是完全随机的，并且符合均值为零的正态分布

8. 对于一个被试团体，实得分数、真分数和测量误差之间的关系

第四章 测量的信度
1. 信度的定义
（1）理论定义
 信度又叫可靠性，指的是多次测量结果的稳定性与一致性程度
（2）操作定义
 ①一组测量分数的真分数变异与总变异（实得分数变异）之比
 ;   
②一个被试团体真分数与实得分数的相关系数的平方
 
 ③一个测验与它的任何一个“平行测验”的相关系数

2. 关于信度
（1）信度说明的是一组测量或一列测量的特性，而不是某个单独测量结果的特性
（2）信度值越接近1，信度越高
（3）信度要求：①标准化或学绩测验的信度在0.9以上
               ②人格测验信度在0.8以上
               ③教师自编学绩测验的信度0.6以上
               ④在实际操作中，标准化测验信度的最低要求至少在0.7以上
（4）不同情况下采用不同方法会得到不同的信度系数
（5）信度系数是对测量结果不一致程度的估计，并没有指出不一致的原因
（6）获得较高的信度系数并不是测量的最终目的
（7）任何测验只能包含特定样本的题目，由特定的施测者，对特定的被试，在特定时间、地点施测，情况不同便会得到不同的分数

3. 信度的作用
（1）反映随机误差大小
（2）解释个人测验分数的意义
（3）比较不同测验分数之间的差异

4. 根据测验的信度和被试实得分数来估计真分数所在置信区间
（1）求出测量的标准误
  
（2）估计真分数所在置信区间
  
 


5. 根据信度可以确定不同测验分数之间是否存在显著差异
（1）将原始分数转换成T分数
\ \ \ \ \ \ T=50+10Z
（2）求出差异的标准误
  
（3）差异的显著性检验

11. 分半法计算同质性信度
（1）斯皮尔曼-布朗校正公式
适用于两半测验的变异数相等的情况
r_{xx}=\frac{2r_{hh}}{1+r_{hh}}
\ \ \ \ \ \ \ \ \ \ \ r_{hh}：两半测验分数的相关系数
（2）弗朗那根公式
     适用于两半测验的变异数不等的情况
r_{xx}=2\left(1-\frac{Sa^2+Sb^2}{S_t^2}\right)
Sa^2：所有被试一半测验分数的方差
Sb^2：所有被试另一半测验分数的方差
S_t^2：所有被试整个测验总分的方差
 
（3）卢仑公式
r_{xx}=1-\frac{S_d^2}{S_t^2}
S_d^2：所有被试两半测验上分数之差的方差
S_t^2：所有被试整个测验总分的方差
 
12. 库德里查森信度系数的计算
适用于二分法计分或0，1计分的题目
r_{xx}=\left(\frac{K}{K-1}\right)\left(\frac{1-K\bar{p\bar{q}}}{S_t^2}\right)
K：测验题目数
\bar{p}：题目的平均通过率
\bar{q}：题目的平均失败率
S_t^2：所有被试整个测验总分的方差
 



13. 克隆巴赫系数的计算
适用于二分法、多重计分的题目
a=\frac{K}{K-1}\left(1-\frac{\sum S_i^2}{S_t^2}\right)
K：测验题目数
S_i^2：所有被试在第i题的分数方差
S_t^2：所有被试整个测验总分的方差

14. 荷伊特信度计算
基本思想：用方差分量比来衡量测验内部一致性
变异的三个来源：被试内变异，题目间变异，人题交互作用变异
r_{xx}=1-MS人×题MS人

15. 分半法的适用条件
（1）适用于难度测验，不适用于速度测验
（2）适用于测验只能施测一次、无复本可用的情况
（3）不适用有任选题（二选一作答）的测验
（4）不适用包含上下有牵连的题的测验

16. 影响信度的因素
（1）被试因素
  ①被试自身引起的误差
  ②被试团体的异质性
  ③被试团体的平均水平
（2）测量工具因素
  ①测验的长度
  ②题目间的同质性程度
（3）施测过程因素

17. 提高信度的方法
（1）适当增加测验的长度
（2）因素分析和鉴别力分析
（3）控制测验题目的难度
（4）选取恰当的被试团体
（5）施测过程的标准化

18. 报酬递减率
r_{kk}=\frac{K\times r_{XX}}{1+\left(K-1\right)r_{XX}}
K：改变长度后的测验与原测验长度之比
r_{XX}：原测验的信度

第五章 测量的效度
1. 效度的定义
（1）理论定义
 又叫有效性，是指一个测验或量表实际能测出其所要测量的心理特质的程度，反映了测量结果的正确性程度
（2）操作定义
 在一列测量中，与测量目的有关的真实变异数与总变异的比率
 

2. 效度的特点
（1）效度说明的是一组或一列测量的特性，而不是某个单独测量的特性
（2）效度是一个相对的概念
（3）效度是测量的随机误差和系统误差的综合反映
（4）判断一个测量是否有效要从多方面收集证据

3. 信度和效度的关系
（1）信度高是效度高的必要而非充分条件
（2）效度受信度的制约：r_{xy}^2\le r_{xx}
4. 效度的种类
效度类型
内容效度 
构想效度 
效标关联效度

第六章 测验的项目分析
1. 测验的项目分析包括难度分析和区分度分析

2. 难度的水平如何确定
①常模参照测验中，项目的难度应尽量接近0.5
大多数项目的难度应在0.3~0.7之间，所有项目的平均难度为0.5，项目的难度分布为正态分布
②标准参照测验或掌握测验中，可不考虑难度
③当测验用于选人时，难度视录取率而定
④难度与测验的性质有关：速度测验难度不宜太高，每个项目难度值相等，难度测验要求难度值在0.5左右

3. 难度的计算
（1）二分法计分项目的难度
①通过率
P=\frac{R}{N}
②高低分组法
P=\frac{1}{2}\left(\frac{R_H}{N_H}+\frac{R_L}{N_L}\right)
 
（2）非二分法记分项目的难度
P=\frac{\bar{X}}{X_{max}}
（3）难度校正
CP=\frac{KP-1}{K-1}
 

4. 难度对测验的影响
（1）难度影响测验分数的分布形态
①极端难度时：分数分布集中在一个分数点上
②难度偏高或偏低时：分数分布呈偏态分布
③难度中等时：分数呈正态分布
（2）难度影响离散程度和信度
①极端难度时：离散程度为0，信度为0
②难度偏高或偏低时：离散程度低，信度低
③难度中等时：信度高
（3）难度影响测验的鉴别力
①极端难度时，分数分布集中在一个点上，没有鉴别力
②难度偏高或偏低时：难度偏高时对高水平被试鉴别力高；难度偏低时对低水平被试鉴别力 高
③难度中等时：鉴别力最高

5. 难度的量表水平：顺序量表

6. 难度的等距变换
（1）Z值越大，难度越高
（2）较难的项目，Z值为正值
（3）较容易的项目，Z值为负值

8. 区分度
又称鉴别力，指测验项目对被试心理品质差异的区分程度或鉴别能力

9. D值的含义
①D=1，高分组全部通过，低分组全部失败，区分度最好
②D>0，高分组通过率大于低分组，有一定区分度
③D=0，高低分组通过率相同，无区分能力
④D<0，高分组通过率低于低分组，题目有问题或答案错

10. D值的分析
①0.4以上：非常优良
②0.3~0.39：良好，如能修改更佳
③0.2~0.29：尚可，仍需修改
④0.19以下：劣，必须淘汰

11. 区分度的计算方法
（1）鉴别指数法（高低分组法）
①二分法记分项目的区分度：D=P_H-P_L=\frac{R_H-R_L}{N}
 
②非二分法记分项目的区分度：D=\frac{{\overline{X}}_H-{\overline{X}}_L}{X_{max}}
 
③独立样本t检验
（2）相关法：项目得分与测验总分的相关
前提条件：①测验效度高  ②排除共变关系
①积差相关
②点二列相关
 
③二列相关
④\varphi相关
（3）项目特征曲线
纵轴：难度
曲线坡度：区分度
 

12. 如何理解区分度的相对性
①用不同的计算方法，所得区分度值不同
②样本容量大小影响相关法区分度值的大小
③分组标准影响鉴别指数
④被试样本的同质性程度影响区分度值的大小

13. 区分度与信度的关系
项目的区分度高，分数的分布范围广，信度高。因此，要提高测验的信度，可先提高项目的区分度。
